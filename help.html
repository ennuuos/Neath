  • How to run from compiled code

    1. Download the compiled code from the compiled branch on github and unzip it.
    2. Run neath.exe

    How to run from source code

    1. Download the source code from github and unzip it. (if you want a specific version change the selected branch or tag)
    2. Download Love2D and unzip it somewhere.
    3. Drag the Neath folder onto love.exe to execute it
